## 📝 Font Recommendations

For the best experience, add this to your settings.json:

```json 
{
  "editor.fontFamily": "'Ubuntu Variable', 'Ubuntu Mono', 'JetBrains Mono', monospace",
  "editor.fontLigatures": true,
  "editor.fontSize": 14,
  "editor.lineHeight": 22
}
```
